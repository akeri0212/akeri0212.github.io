History

  • 1985年 誕生〜小学生 小さい頃から工作やお絵かきなど「作ること」が好きでした。
  • 1997年 中学生 家のパソコンに入っていた photoshop で遊びながら操作を覚えました。 学校では入学式の演奏に感動し、吹奏楽部に入部。以来フルートは人生をかけた趣味となりました。
  • 2000年 高校生 携帯電話をゲット。初のカラー液晶端末で、カラフルに動くチェーンメールに興味を持ちました。転送画面で表示されるタグが、HTMLとの出会いでした。 どのタグでどんな効果があるのか研究して、自分でも使ってみました。
  • 2003年 大学生 3Dアニメーションをやってみたくて、クリエイティブを学ぶことが出来る大学に進学。趣味のイラストサイトを作ったのもこの頃。 結局「何かやりたい事と違う」と感じて3Dはほとんどやりませんでしたが、授業で出会った「ノンデザイナーズ・デザインブック」で、「芸大でデザインを学んで無くてもデザインって出来るんだ！」とデザイナーの仕事に興味をつキッカケになりました。
  • 2007年 web制作会社へ入社 自身でサイト制作を経験したこともあり、web制作会社に応募して就職が決定。 初めはデザイナーとして勤務したものの、やはり現実は厳しく苦戦しました。並行して学習していたフロントエンド設計の方がすんなり理解出来る感覚があり、デザインもある程度出来るようになりましたが4年ほどハイブリットに経験した後はフロントエンドに注力するようになりました。 4年目には制作チームのひとつをリーダーとして任せていただき、人材育成、プロジェクト管理等のマネジメントを経験しました。
  • 2011年 社会人の吹奏楽バンドで出会った夫と結婚
  • 2013年 連日12時間以上の勤務が続いて疲れを感じていた頃、証券会社への出向が決定。 出向先は定時退社できましたが、オペレーター業務を淡々とこなす日々でした。
  • 2014年 産休〜第一子出産〜育休 一年程お休みをいただき、育児に専念しました。 一度仕事から離れたことでリフレッシュし、改めて「仕事が好き」と認識できました。
  • 2015年 時短で職場復帰 やはりひとりでフロントエンド設計を完結させるのは難しく、教育も兼ねて新人と分担し、コードレビューも行いました。
  • 2017年 第二子出産
  • 2018年 二度目の職場復帰 後輩のサポート業務が多く、時短勤務に疑問を持ち始める。 良いチームメンバーに恵まれ、経験値を元に手助けできる充実感もあったが、転職を決意。
  • 2019年 地元の求人広告代理店へ転職 自宅から近く、通勤時間を短縮できることが決め手で入社。 エンジニアチームの無い環境、自社サービスの開発、運用、バックエンドエンジニア業務など、前職で経験の無かったことばかりで毎日学びの連続です。
  • 現在 予定より早くフルタイム勤務希望となったものの現職では難しい事、 技術的な相談ができるメンバーがいない事により、新しい勤務先を探しています。
